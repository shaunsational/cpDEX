                      About

                      This project started because I was tired of looking up the level 30 cp for a pokemon every community day to see what the hundo would be. I swore there was a site that used to display them the way they are here. I also thought someone would have compiled a JSON file with most of the info needed. Apparently I was wrong. So about a month later, and many additions later, here's the result.

                      • GO Hub for most of the pokémon data
                      • LeekDuck for event and shiny information
                      • Phoenix Code for the example that kickstarted this layout, and allowing developers to extend the functionality of their editor via plugins.
